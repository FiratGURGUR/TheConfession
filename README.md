# TheConfession

Basit bir sosyal medya uygulaması geliştiriyorum.

## Kullandığım teknolojiler:

- Dagger2
- Mvvm
- DataBinding
- Medusa
- Retrofit
- RxJava

Backend kısmını php ile geliştiriyorum ve mySql kullanıyorum.
